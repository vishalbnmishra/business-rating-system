#  Business Rating System

## Project Overview

Business Rating System is a web-based application developed using Core PHP and MySQL that allows users to manage businesses and submit ratings dynamically.  

The system supports full CRUD operations and calculates average ratings in real time without requiring a page reload using AJAX.

This project demonstrates backend logic handling, database relationships, asynchronous requests, and responsive UI implementation.

---

##  Features

- Add, Edit, and Delete Businesses (CRUD Operations)
- Submit ratings for businesses
- Real-time average rating calculation
- Dynamic dropdown updates without page refresh
- AJAX-based form submissions
- Responsive design using Bootstrap
- Star rating UI using jQuery Raty plugin
- Server-side validation and structured database queries
